Flag 0 Guide

Target Hint 0 Check the Letter of Engagement for any websites.
Target Hint 1 The ACME Company web page is linked in the footer of the letter of engagement.
Target Solution You must try to break into the customer dashboard of ACME company. This is located at /acme-company/customer-dashboard/
Flag Hint 0 The ACME Company customer dashboard requires a user and password.
Flag Hint 1 Check to see if there are any other files in the /acme-company/ web server directory.
Flag Hint 2 The web server lists all files in the /acme-company/ directory. Simply navigate to /acme-company/. Do you see any files that look interesting?
Flag Hint 3 You can see the /acme-company/customer-dashboard.htpasswd file is visible and available to download. This is a password hash file. Perhaps it contains the passwords to something important. See if you can crack it.
Flag Hint 4 The /acme-company/customer-dashboard.htpasswd file can be cracked using John the Ripper. We can see that it contains an entry for a user named admin.

To crack with John the Ripper we need a word list. The cewl tool can be used to do this. Scan the ACME Company home page and make it a word list: 
							  
cewl --write ctf-wordlist.txt http://192.168.20.20:1111/acme-company/
This will generate a word list based off of the words on the ACME company web page. Next use John the Ripper on this downloaded htpasswd file with our word list: 
							  
john --wordlist ctf-wordlist.txt customer-dashboard.htpasswd
Running John the Ripper will give you the login information.
Flag Solution The login to /acme-company/customer-dashboard/ is username admin with password money.
